An automated bot for interacting with the Ekox protocol on Hoodi network, supporting deposit, withdraw, and claim operations.

## Features

- **Multi-Wallet Support**: Manage multiple wallets simultaneously
- **Deposit ETH**: Stake ETH to receive exETH
- **Withdraw exETH**: Convert exETH back to ETH
- **Claim**: Claim unlocked withdrawals
- **Daily Run Mode**: Automated cycling through deposit, withdraw, and claim operations

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Private keys for wallets you want to use
- ETH on Hoodi network
